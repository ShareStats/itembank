    Question

    In een onderzoeksproject vergeleken onderzoekers twee bijlesprogramma’s. Studenten werden willekeurig toegewezen aan een van de twee bijlesprogramma’s.
    De onderstaande tabel laat zien:

    Studenten die bijles-programma “Boost” kregen scoorden gemiddeld een cijfer van 1,5 punten hoger dan voordat ze aan het bijles-programma begonnen, met een standaarddeviatie van 0,5.

    Studenten die bijles-programma “Enhance” kregen, scoorden gemiddeld een cijfer van 1,0 punt hoger dan voordat ze met het bijles-programma begonnen, met een standaarddeviatie van 1,0.

    In welk programma was er meer variatie in de stijging van de cijfers?
     


    1. FALSE: In het bijles-programma “Enhance”, want de standaarddevatie ligt daarbij dichter bij de gemiddelde score.
    2. FALSE: In het bijles-programma “Boost”, want de standaarddeviatie is daar kleiner.
    3. TRUE: In het bijles-programma “Enhance”, want de standaarddeviatie is daar groter.
    4. FALSE: In het bijles-programma “Boost”, want de standaarddevatie ligt daarbij dichter bij de gemiddelde score.

    1. False
    2. False
    3. True
    4. False
