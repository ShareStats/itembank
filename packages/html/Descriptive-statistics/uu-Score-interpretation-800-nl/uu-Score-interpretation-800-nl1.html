    Question

    Een onderzoeker is geïnteresseerd in verschillen in de hartslag van mensen in angstige situaties. Hij onderscheidt daarbij twee groepen: mensen met een fobie en mensen zonder fobie. De onderzoeker wil een covariantieanalyse uitvoeren waarin gecontroleerd wordt voor hartslag in rust (hartslag_rust). Voorafgaand controleert hij of de aanname van homogene regressie al dan niet is geschonden. De SPSS-uitvoer van deze analyse is hieronder weergegeven.

    Is voldaan aan de assumptie van homogene regressie (α\alpha = .05 = 10%) en kan de onderzoeker doorgaan met het uitvoeren van een ANCOVA?


    1. FALSE: Er is voldaan aan de assumptie, want p < .10, dus een ANCOVA kan niet worden uitgevoerd.
    2. TRUE: Er is voldaan aan de assumptie, want p = .22, dus een ANCOVA kan worden uitgevoerd.
    3. FALSE: De assumptie is geschonden, want p < .10, dus een ANCOVA kan worden uitgevoerd.
    4. FALSE: De assumptie is geschonden, want p = .22, dus een ANCOVA kan niet worden uitgevoerd.
