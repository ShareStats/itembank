    Question

    Welke stelling(en) is/zijn correct over de regressievergelijking Y = b0 + b1X?

    I. Y staat eigenlijk voor de voorspelde waarde door de regressie.
    II. b0  is het gemiddelde waarde van Y als X gelijk is aan 0


    1. FALSE: Stelling I. is correct en stelling II. is niet correct
    2. FALSE: Stelling I. is niet correct en stelling II. is correct
    3. TRUE: Beide stellingen zijn correct
    4. FALSE: Beide stellingen zijn niet correct

    Solution

    Stelling I is juist omdat de regressievergelijking waarden van Y voorspelt door middel van de regressielijn a + b1*X.
    Stelling II is ook juist omdat als je X vervangt door 0 dan wordt de vergelijking Y=b0. Dit betekent dat de voorspeelde waarde van Y (of anders: het gemiddelde Y) gelijk aan de constante is.


    1. False
    2. False
    3. True
    4. False
