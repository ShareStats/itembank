    Question

    Hieronder een deel van de SPSS uitvoer van een statistische analyse naar het effect van het boek dat gebruikt wordt bij een statistiekcursus (BOEK met 1 = Statistiek voor dummies en 2 = Discovering statistics) en het type onderwijs dat gegeven wordt (TYPEOW met 1 = hoorcollege, 2 = werkgroep, 3 = practicum) op de tevredenheid van studenten (Tevredenheid gemeten op een schaal van 0 tot 10).

    Uitgaande van de onderstaande resultaten, welke van de onderstaande uitspraken is dan correct als we toetsen bij Î±\alpha = 1%?


    1. FALSE: Er is een significant hoofdeffect van het boek dat gebruikt wordt op tevredenheid.
    2. FALSE: Er is geen significant hoofdeffect van type onderwijs op de tevredenheid.
    3. FALSE: Voor het type onderwijs is er een significant verschil in het effect van het boek op tevredenheid.
    4. TRUE: Voor de twee boeken is er geen significant verschil in het effect van type onderwijs op tevredenheid.
