    Question

    In een onderzoek naar invloeden op de genezing na een ingrijpende operatie, worden de volgende variabelen betrokken: ziekte-inzicht en IQ. Bij een mediator analyse waarbij wordt onderzocht of de invloed van IQ verloopt via ziekte-inzicht, vindt de onderzoeker de volgende verbanden (uitgedrukt in regressie coëfficiënten):

    1. b = .64 van IQ -> genezing.
    2. b = .56 van IQ -> ziekte-inzicht.
    3. b = .55 van ziekte-inzicht -> genezing (waarbij IQ ook als predictor is opgenomen in het model)

    De laatste analyse in deze mediator analyse is een regressie analyse met IQ en ziekte-inzicht als predictoren en genezing als afhankelijke variabele…


    1. TRUE: waarbij je verwacht dat regressie coëfficiënt (1) zal dalen.
    2. FALSE: waarbij je verwacht dat regressie coëfficiënt (2) zal dalen.
    3. FALSE: waarbij je verwacht dat regressie coëfficiënt (3) zal dalen.
    4. FALSE: waarbij je verwacht dat alle drie de regressie coëfficiënten zullen dalen.

    Solution

    Om mediatie te toetsen onderzoeken we de vier stappen (zie vraag 12). De laatste stap is kijken of de relatie tussen IQ en genezing nog steeds significant is, wanneer er gecontroleerd is voor ziekteinzicht?

    Wanneer er sprake is van mediatie zal de relatie tussen IQ en genezing minder sterk zijn wanneer ook ziekteinzicht in de analyse is opgenomen. Je verwacht dus dat regressie coëfficiënt (1) zal dalen.
