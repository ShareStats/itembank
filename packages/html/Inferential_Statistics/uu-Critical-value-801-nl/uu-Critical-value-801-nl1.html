    Question

    Om te toetsen of de CITO-toetsscore een significante voorspeller is van het gemiddelde cijfer aan het einde van de brugklas, kunnen we de volgende hypothesen toetsen: H0: β\beta = 0 H1: β\beta ≠ 0 Wat is het resultaat van deze toets bij α\alpha = .05?


    1. TRUE: CITO-toets score is een significante voorspeller, want de toetsingsgrootheid is groter dan de kritieke waarde.
    2. FALSE: CITO-toets score is geen significante voorspeller, want de toetsingsgrootheid is kleiner dan de kritieke waarde.
    3. FALSE: CITO-toets score is geen significante voorspeller, want de p-waarde is kleiner dan .05.
    4. FALSE: Om te bepalen of CITO-toets score een significante voorspeller is, hebben we de gehele SPSS-output nodig; het kan aan de hand van deze gegevens niet bepaald worden.
