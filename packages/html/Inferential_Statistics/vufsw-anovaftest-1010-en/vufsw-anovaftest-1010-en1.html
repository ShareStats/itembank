    Question

    What is the missing F-value of GROUP in the ANOVA table that is presented below? Round to three decimals.


    Solution

    SSM / dfM = MSM
    SSR / dfR = MSR
    MSM / MSR = F

    MSM = 5.624 / 3 = 1.875
    MSR = 57.695 / 95 = 0.607
    F = 1.875 / .607 = 3.089
