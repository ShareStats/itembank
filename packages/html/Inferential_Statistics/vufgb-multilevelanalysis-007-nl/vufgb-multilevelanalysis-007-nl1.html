    Question

    Een hoogleraar doet onderzoek naar het effect van een leerkrachtinterventie op het verminderen van ADHD-symptomen van kinderen met ADHD. Zij heeft ADHD-symptomen herhaaldelijk gemeten tijdens en na de interventie. Zij vermoedt dat de interventie niet bij alle kinderen hetzelfde effect heeft: bij sommige kinderen verminderen de ADHD-symptomen sterk, bij andere kinderen veel minder.

    Met welk model kan zij deze hypothese toetsen?


    1. FALSE: Herhaalde metingen ANOVA met lineair effect van tijd.
    2. FALSE: Herhaalde metingen ANOVA met kwadratisch (‘cubic’) effect van tijd.
    3. FALSE: Linear mixed model met random intercept en fixed slope.
    4. TRUE: Linear mixed model met random intercept en random slope.

    1. Incorrect

    4. Correct
