    Question

    Which of the following statements is true about MANOVA

    I. MANOVA controls for the overall error rate, ensuring that the risk of Type I errors does not increase.
    II. MANOVA is suitable to study the effect of an experimental manipulation on dependent variables which are not correlated with each other.


    1. TRUE: Statement I is correct and statement II is incorrect.
    2. FALSE: Statement I is incorrect and statement II is correct.
    3. FALSE: Both statements are correct.
    4. FALSE: Both statements are incorrect.

    Solution

    By analyzing simultaneously several dependent variables MANOVA corrects for the familywise error rate. Thus, the statement “It controls for the overall error rate, ensuring that the risk of Type I errors does not increase.” is correct.

    MANOVA studies the effect of (at least one) independent variable on several dependent variables that are correlated with each other. If they are uncorrelation with each other, MANOVA makes no sense. therefore the statement" It is suitable to study the effect of an experimental manipulation on dependent variables which are not correlated with each other." is incorrect


    1. True
    2. False
    3. False
    4. False
