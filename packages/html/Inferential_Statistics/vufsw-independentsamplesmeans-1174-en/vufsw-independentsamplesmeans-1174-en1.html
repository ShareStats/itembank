    Question

    With two brands of candles (Brand X and Brand Y), it has been tested how many hours they will last. The results are below.

    H0: mean1 - mean2 = 0
    Ha: mean1 - mean2 is not equal to 0

    Draw conclusions about the hypotheses with a significance level of .01 (alpha = 1%).


    1. TRUE: We can reject the null hypothesis, since the absolute observed t-value is greater than the critical t-value.
    2. FALSE: We can not reject the null hypothesis, since the absolute observed t-value is greater than the critical t-value.
    3. FALSE: We can reject the null hypothesis, since the absolute observed t-value is less than the critical t-value.
    4. FALSE: We can not reject the null hypothesis, since the absolute observed t-value is less than the critical t-value.

    Solution

    We can reject the null hypothesis, since the absolute observed t-value is greater than the critical t-value.
    We can calculate the t value with:

    Compare means: (ȳ2 - ȳ1) ± t(se) with se = sqrt [ (s12 / n1) + (s22 / n2) ]
    Test H0 : μ1 = μ2 using t = (ȳ2 - ȳ1) / se
