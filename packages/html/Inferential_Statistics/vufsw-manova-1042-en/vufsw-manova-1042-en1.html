    Question

    To prefer a MANOVA over separate ANOVA’s, the dependent variables should …


    1. FALSE: Weakly correlate.
    2. TRUE: Moderately correlate.
    3. FALSE: Perfectly correlate.
    4. FALSE: Very strongly correlate.

    Solution

    The intuition is that since MANOVA accounts for the correlation between the dependent variables, then if the dependent variables are not correlated then there is no need for a MANOVA and you can run simple ANOVAs. If there is a very high correlation, then by explaining the variance in one of the dependent variables, you explain what you can explain with this model. So, again there is no need for a MANOVA.


    1. False
    2. True
    3. False
    4. False
