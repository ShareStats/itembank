    Question

    Bereken het aantal vrijheidsgraden behorend bij het Interactie-effect AB in een factoriele ANOVA met 3 X 3 design.


    1. FALSE: 2
    2. FALSE: 3
    3. TRUE: 4
    4. FALSE: 9

    Solution

    df=(g−1)×(h−1)=(3−1)×(3−1)=2×2=4df = (g-1) \times (h-1) = (3-1) \times (3-1) = 2 \times 2 = 4.

    Als je ANOVA model als regressiemodel met dummy variabelen uitschrijft krijg je twee dummy variabelen voor Factor 1 en twee dummy variabelen voor Factor 2. Dat levert in het regressiemodel 2×2=42 \times 2 = 4 interactietermen op. Verschil in parameters tussen model met en zonder interactie is dan dus 4.

    Dit is de redenering als je de F-toets opvat als model vergelijkingstoets!

    2. Incorrect
    3. Correct
