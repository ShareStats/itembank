    Question

    In a regression analysis to predict dependent variable Y from predictor X, the linearity assumption is being checked. The following output is available:


    1. FALSE: F-change is significant, so the linearity assumption is satisfied.
    2. TRUE: F-change is significant, so the linearity assumption is not satisfied.
    3. FALSE: F-change is not significant, so the linearity assumption is satisfied.
    4. FALSE: F-change is not significant, so the linearity assumption is not satisfied.

    Solution

    F-change (comparing model 1 to model 2) is significant, indicating that adding x-squared is a significant improvement to the model and therefore the linearity assumption is not satisfied.
