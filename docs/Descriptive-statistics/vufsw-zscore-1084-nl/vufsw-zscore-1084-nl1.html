    Stel dat een van de bezoekers een lichamelijke hygiënescore van 0,88 heeft op dag drie. Wat is zijn/haar gestandaardiseerde score (z-score)?


    1. TRUE: -0.14
    2. FALSE: -0.28
    3. FALSE: -3.38
    4. FALSE: -4.01
    5. FALSE: -1.24

    Solution

    De formule is: z = (waarneming - gemiddelde) / standaardafwijking Dus z = (0,88 - 0,98) / 0,71 = -0,14
