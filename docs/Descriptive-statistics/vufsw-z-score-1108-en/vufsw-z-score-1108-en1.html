    Question

    We derived the following figures for the variable ‘age’ from a random sample: number of respondents 200, mean 46.1340, standard error 0.11751, standard deviation 1.66179.

    What is the z-score of a 44 year old person?


    1. TRUE: (44-46.1340)/1.66179
    2. FALSE: (46.1340-44)/1.66179
    3. FALSE: (44-46.1340)/0.11751
    4. FALSE: (44-46.1340)/2.762

    Solution

     formula z = (x - μ) / σ where x is your data point, μ is the mean, and σ is the standard deviation


    1. True
    2. False
    3. False
    4. False
