    Question

    We hebben de volgende cijfers voor de variabele ‘leeftijd’ uit een aselecte steekproef afgeleid steekproef: aantal respondenten 200, gemiddelde 46,1340, standaardafwijking 0,11751, standaardafwijking 1,66179.

    Wat is de z-score van een 44-jarige?


    1. TRUE: (44-46.1340)/1.66179
    2. FALSE: (46.1340-44)/1.66179
    3. FALSE: (44-46.1340)/0.11751
    4. FALSE: (44-46.1340)/2.762
