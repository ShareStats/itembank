    Question

    Er zijn 151 ongevallen tijdens sportlessen op middelbare scholen. 54 van deze ongevallen werden veroorzaakt door cheerleaden. Stel dat je een random steekproef van 5 uit de 151 wil nemen zonder terugleggen. Wat is dan de kans dat er geen enkel ongeluk veroorzaakt wordt door cheerleaden?


    1. FALSE: (49/151)5
    2. FALSE: 1-(54/151)5
    3. FALSE: (49/151)(48/150)(47/149)(46/148)(45/147)
    4. FALSE: (49/151)(48/151)(47/151)(46/151)(45/151)
    5. TRUE: (97/151)(96/150)(95/149)(94/148)(93/147)

    Solution

    Het correcte antwoord is:

    • (97/151)(96/150)(95/149)(94/148)(93/147)
