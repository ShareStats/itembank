    Question

    Een fabriek in Arnhem produceert spaarlampen. De kans op een defect is 5%. Bij een steekproef pakken we willekeurig twee lampen met terugleggen. In de volgende tabel zien we dat het aantal lampen X dat wordt gepakt de volgende kansverdeling heeft:

    Waarde van X 0 1 2
    kans 0,9025 0,095 ?

    Hierbij is X een toevalsvariabele. X = 0 betekent dat de twee lampen niet defect zijn; X = 1 betekent dat er één van de twee defect is; X = 2 betekent dat de twee lampen allebei defect zijn. Wat is de kans dat beide lampen defect zijn? Rond af op vier decimalen.


    Solution

    The correct answer is 0.0025.
