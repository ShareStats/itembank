Het winnende lot levert de winnaar €; 750.000 op. De kans dat men een winnend lot trekt, bij aanschaf van één lot, is 0,000001. De kansverdeling voor X (= het geldbedrag dat men ontvangt) ziet er als volgt uit:

    Prijs X € 0 € 750.000
    Kans 0,999999 0,000001

    Een lot kost dit jaar €; 2,50. Hoeveel euro incasseert de Belgische staat op de lange termijn per lot? Kies uit onderstaande mogelijkheden.


    1. FALSE: € 0,75
    2. FALSE: € 1,25
    3. TRUE: € 1,75
    4. FALSE: € 1,95
