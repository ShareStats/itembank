    Question

    Rechts zie je een Venn-diagram dat de relatie representeert tussen twee gebeurtenissen, namelijk A en B. Welke onderstaande situatie past hierbij het beste?


    1. TRUE: A is rode thee, B is groene thee, C is alle theesoorten.
    2. FALSE: A is zoetwatervissen, B is vissen, C is de familie der vissen.
    3. FALSE: A is {1,2,3,5}, B is {2,3,6,7}, C is{1,2,3,5,6,7}.
    4. FALSE: A is vrouwen, B is mannen, C is bejaarden.

    Solution

    Het correcte antwoord is:

    • A is rode thee, B is groene thee, C is alle theesoorten.
