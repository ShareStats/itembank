    Question

    When does one call two events disjoint?


    1. FALSE: If the two events have joint outcomes.
    2. TRUE: If the two events do not have joint outcomes.
    3. FALSE: If the two events are independent of each other.
    4. FALSE: None of the above answers is correct.

    1. If the two events have joint outcomes: Incorrect
    2. If the two events do not have joint outcomes: Correct
    3. If the two events are independent of each other: Incorrect
    4. None of the above answers is correct: Incorrect
