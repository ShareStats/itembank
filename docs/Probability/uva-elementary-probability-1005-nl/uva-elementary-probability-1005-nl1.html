    Question

    Een schroefkop kan verschillende vormen hebben: (a) zeskant, (b) bol, (c) verzonken, (d) cilinder, (e) verzonken met binnenzeskant, (f) verzonken met kruis, (g) kartelschroef. Een fabriek heeft 1000 schroefkoppen gemaakt van iedere vorm. We noemen de gebeurtenissen:

    * A = {de schroeven met bolkop}
    * B = {de schroeven met verzonken kop}
    * C = {de schroeven met verzonken met binnenzeskant-kop}

    Wat is dan de kans op A U B?


    1. FALSE: 1/7000
    2. FALSE: 1/1000
    3. FALSE: 1/7
    4. TRUE: 2/7

    Solution

    Het correcte antwoord is:

    • 2/7
