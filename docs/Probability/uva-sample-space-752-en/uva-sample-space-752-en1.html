    Question

    Genetics teaches that children inherit genes from their parents. If both parents carry the genes O and A for blood type, each child has a chance of 0.25 to inherit two O genes to become blood type O. The number of children with blood type O among five offspring of these parents is the number of successes X in five independent attempts. Suppose that the random variable X is the number of children with blood type O. What are then the possible values for X?


    1. FALSE: A, B, AB and O
    2. TRUE: 0, 1, 2, 3, 4 and 5
    3. FALSE: 1, 2, 3, 4 and 5
    4. FALSE: (5, 0,25)

    1. A, B, AB and O: Incorrect
    2. 0, 1, 2, 3, 4 and 5: Correct
    3. 1, 2, 3, 4 and 5: Incorrect
    4. (5, 0,25): Incorrect
