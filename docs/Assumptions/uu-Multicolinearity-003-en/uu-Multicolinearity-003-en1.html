    Question

    What does multicollinearity mean?


    1. FALSE: There is a linear relationship between the independent and dependent variable.
    2. TRUE: There is a high correlation between two or more independent variables.
    3. FALSE: The independent variables are not correlated.
    4. FALSE: The dependent variable is affected by variables not included in the model.
