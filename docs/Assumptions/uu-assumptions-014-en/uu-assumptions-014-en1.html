    Question

    What is no assumption in multiple linear regression?


    1. TRUE: Dependent observations
    2. FALSE: Homoscedasticity
    3. FALSE: Normally distributed errors
    4. FALSE: Linear relationships between the Xs and Y

    Solution

    An assumption in multiple linear regression is independent observations. Homoscedasticity, normally distributed errors and linear relationships between the Xs and Y are all assumptions in multiple linear regression.
