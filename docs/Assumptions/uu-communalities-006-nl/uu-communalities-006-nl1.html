    Question

    Een onderzoek naar internaliserende en externaliserende probleemgedragingen (X1 tot en met X5) bij plattelandsjongeren resulteerde in de volgende correlatiematrix:

    De communaliteit h2 van variabele X5 (PSYCHOSOMATISCH) is gelijk aan .61.

    Van welke van hier onder genoemde probleemgedragingen is de verklaarde variantie van de twee-factoroplossing hoger?


    1. TRUE: X1 criminaliteit
    2. FALSE: X2 vandalisme
    3. FALSE: X3 depressie
    4. FALSE: X4 eetstoornissen

    Solution

    Uitleg: Om de verklaarde variantie uit te rekenen van elke variabele, dient de communaliteit per variabele berekent te worden. Hiervoor wordt de volgende formule gebruikt:

    hg2=∑agt2,h^2_g = \sum a^2_{gt} , Deze formule geeft aan dat de communaliteit van een variabele de som van de gekwardrateerde factorladingen wordt gebrukt. Hieronder wordt dit gedaan voor de vier variabelen:

    hx1criminaliteit2=.672+.532=.449+.281=.73h^2_{x1criminaliteit} = .67^2 + .53^2 = .449+ .281 =.73 hx2vandalisme2=.632+.172=.397+.004=.40h^2_{x2vandalisme} = .63^2 + .17^2 = .397 + .004 = .40 hx3depressie2=.762+−.132=.578+.017=.60h^2_{x3depressie} = .76^2 + -.13^2 = .578 + .017 =.60 hx4eetstoornissen2=.672+−.212=.449+.044=.49h^2_{x4eetstoornissen} = .67^2 + -.21^2 = .449 + .044 = .49
