    Question

    Welke van de onderstaande uitspraken over ‘kwantitatieve’ variabelen zijn juist?
     
    I.   Dichotoom = de classificatie bestaat uit twee elkaar uitsluitende en uitputtende (exhaustive) categorieën.
    II.  Nominaal = de classificatie bestaat uit meerdere elkaar uitsluitende en uitputtende (exhaustive) categorieën.
    III.  Ordinaal = de classificatie bestaat uit elkaar uitsluitende en uitputtende categorieën en heeft een rangordening.
    IV.  Ratio / Interval = de classificatie bestaat uit elkaar uitsluitende en uitputtende categorieën, heeft een rangordening, en heeft gelijke afstanden tussen de gemeten categorieën.


    1. TRUE: Alle uitspraken zijn juist.
    2. FALSE: Uitspraak I. en II. zijn juist.
    3. FALSE: Uitspraak II., III., en IV. zijn juist.
    4. FALSE: Uitspraak III. en IV. zijn juist.

    1. True
    2. False
    3. False
    4. False
