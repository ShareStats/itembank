    Question

    Welke van de volgende variabelen is geen continue variabele?


    1. FALSE: De tijd benodigd om een som op te lossen.
    2. FALSE: Temperatuur.
    3. FALSE: Gewicht.
    4. TRUE: Aantal kinderen per gezin.

    1. Tijd is een continue variabele. 3,5 minuten of 6,93 seconden zijn mogelijke waarden.
    2. Temperatuur is een continue variabele. Het kan 13,5 graden zijn buiten of de oven kan 181,7 graden warm zijn.
    3. Gewicht is een continue variabele. Iemand kan 71,7 kilo wegen of je kunt 145,6 gram suiker over hebben in de zak.
    4. Dit antwoord is correct. We tellen het aantal kinderen. Dit is een discrete variabele.
