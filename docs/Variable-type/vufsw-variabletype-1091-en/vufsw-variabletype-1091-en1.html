    Question

    A lecturer wants to know whether the average examination grades of two different classes differ from each other because they followed education in a different way. One group received interactive education, the other did not. The two classes are indicated by the variable ‘Interactive class’ (yes / no). In this case:


    1. FALSE: The average exam grade is the independent variable and the number of students in the interactive class is the dependent variable.
    2. FALSE: The average exam grade is the independent variable and the variable ‘Interactive class’ the dependent variable.
    3. FALSE: The number of students in the interactive class is the independent variable and the average exam grade is the dependent variable.
    4. TRUE: The variable ‘Interactive class’ is the independent variable and the average exam grade is the dependent variable.

    Solution

    The variable ‘Interactive class’ is the independent variable and the average exam grade is the dependent variable.
