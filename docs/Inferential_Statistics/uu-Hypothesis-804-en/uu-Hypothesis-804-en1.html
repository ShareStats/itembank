    Question

    One wants to see if there is a significant correlation between number of times exercising per month and heart rate (at rest) in 19 subjects. One-way testing is performed at 5%. The percentage of variance in Y explained by the variance in X is 16%.

    Should the hypothesis that there is a correlation be accepted?


    1. FALSE: No, because robt < rcrit.
    2. FALSE: No, because robt >= rcrit.
    3. TRUE: Yes, because robt < rcrit.
    4. FALSE: Yes, because robt => rcrit.
