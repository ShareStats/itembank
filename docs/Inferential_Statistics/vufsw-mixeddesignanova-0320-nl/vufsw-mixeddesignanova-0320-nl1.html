    Question

    Je onderzoekt of het bijwonen van een eredivisie voetbalwedstrijd een effect heeft op de mate van eigen sportbeoefening. Je bekijkt ook of dit effect verschilt tussen échte voetbal fans en niet echte voetbalfans. Je trekt een willekeurige steekproef van échte voetbalfans en een andere willekeurige steekproef van niet echte voetbalfans. Daarna meet je voor elk individu hun lichaamsbeweging vóór het bijwonen van de voetbalwedstrijd, de dag ná het bijwonen van de voetbalwedstrijd en 3 maanden na het bijwonen van de voetbalwedstrijd. Andere kenmerken (e.g. opleidingsniveau, leeftijd) van het individu zijn ook beschikbaar.

    Welk statistisch model is correct voor deze situatie?


    1. FALSE: One-way ANOVA
    2. FALSE: Regressie met een moderatie-effect (interactie)
    3. FALSE: One-way Repeated Measures ANOVA
    4. TRUE: Mixed-design ANOVA

    1. False
    2. False
    3. False
    4. True
