    Question

    Hieronder staat de formule van een enkelvoudig regressiemodel van Y.
     **
     **Y = a + b1 * X1 

    Wat betekent de “a” in de formule? En welke waarde neemt deze aan?


    1. FALSE: Dit is het significantieniveau, deze waarde bepaalt de onderzoeker zelf en is meestal 0.05.
    2. FALSE: Dit is het intercept, deze waarde is 0 als X1 geen effect heeft op Y.  
    3. TRUE: Dit is de constante, dit is de waarde van Y als X1 gelijk is aan 0.  
    4. FALSE: Dit is de alpha, deze waarde hoort bij een bepaalde t-statistic en moeten we opzoeken in een tabel.
    5. FALSE: Dit is de waarde van de alternatieve hypothese (Ha), deze waarde wordt berekend door SPSS op basis van de steekproefgegevens.

    1. False
    2. False
    3. True
    4. False
    5. False
