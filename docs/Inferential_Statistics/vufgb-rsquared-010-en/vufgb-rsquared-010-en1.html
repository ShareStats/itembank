    Question

    A multiple regression analysis is executed in SPSS.

    Underneath you will find the incomplete ANOVA table.

    Calculate the R-squared (R2R^{2}) of the model.


    1. FALSE: 0.33
    2. FALSE: 0.50
    3. FALSE: 0.60
    4. TRUE: 0.67

    Solution

    The formula for the R2R^{2} is (TSS−SSE)TSS\frac{(TSS-SSE)}{TSS}. The SSE=MSE×df=10×20=200SSE = MSE \times df = 10 \times 20 = 200.

    The corresponding R2R^{2} is (600−200)600=400600=0.67\frac{(600-200)}{600} = \frac{400}{600} = 0.67. Notice the 400 corresponds the the Regression Sum of Squares, which can also be determined by calculating 250×1+150×1250 \times 1+150 \times 1.
