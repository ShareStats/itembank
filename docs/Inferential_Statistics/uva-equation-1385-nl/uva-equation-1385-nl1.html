    Question

    Vetpercentage kan geschat worden met behulp van (de logaritme van) de som van de huidplooidiktes van biceps en triceps.
    Bijgaand is een spreidingsdiagram waarin deze beide variabelen tegen elkaar staan uitgezet.
    De vergelijking van de bijbehorende regressielijn luidt:


    1. FALSE: vetpercentage = 5 + 24 x log(huidplooidikte)
    2. FALSE: vetpercentage = 9 + 19 x log(huidplooidikte)
    3. FALSE: vetpercentage = -13 + 25 x log(huidplooidikte)
    4. TRUE: vetpercentage = -6 + 17 x log(huidplooidikte)

    1. vetpercentage = 5 + 24 x log(huidplooidikte): Incorrect
    2. vetpercentage = 9 + 19 x log(huidplooidikte): Incorrect
    3. vetpercentage = -13 + 25 x log(huidplooidikte): Incorrect
    4. vetpercentage = -6 + 17 x log(huidplooidikte): Correct
