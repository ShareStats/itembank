    Question

    Which of the following is a test that can be used to assess multivariate differences in MANOVA?


    1. FALSE: Roy’s largest root
    2. FALSE: Wilks’s lambda
    3. TRUE: All other answers are correct
    4. FALSE: Hotelling’s T2

    Solution

    Hotelings’ T2, Roy’s largest root and Wilk’s Lamda are all tests for multivariate differences in MANOVA. Therefore, the option ‘All the other answers are correct’ is the correct one.


    1. False
    2. False
    3. True
    4. False
