Het percentage dat met "ja" antwoordt is 25% uit een steekproef van 100 aselect gevraagde mensen. De onderzoekers willen weten of het percentage groter is dan 20%. Daartoe stellen zij de nulhypothese op H0: p0 = 20%. Vanaf welk percentage kan de nulhypothese worden verworpen wanneer een significantieniveau wordt gebruikt van 2.5%?

    3. TRUE: 0.28
