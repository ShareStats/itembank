    Question

    Als een toets wordt uitgevoerd met Î±=5\alpha = 5% en een power heeft van 8080%, wat is dan de kans dat de H0H_0 terecht wordt verworpen?


    1. FALSE: 5%
    2. FALSE: 20%
    3. TRUE: 80%
    4. FALSE: 95%

    1. 5% is de kans dat H0H_0 wordt verworpen terwijl deze juist is (kans op een Type I fout).
    2. 20% is de kans dat H0H_0 niet wordt verworpen terwijl deze wel verworpen had moeten worden (kans op een Type II fout).
    3. 80% is de kans dat er in de steekproef een effect gevonden wordt, wanneer deze in de populatie ook daadwerkelijk aanwezig is. Dit is dus de kans H0H_0 wordt verworpen terwijl deze ook niet juist is (en dus terecht verworpen wordt).
    4. 95% is de kans dat H0H_0 niet wordt verworpen wanneer deze ook daadwerkelijk juist is.
