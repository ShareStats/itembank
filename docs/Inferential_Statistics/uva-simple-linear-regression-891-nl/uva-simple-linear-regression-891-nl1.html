De coëfficiënt is b1 = 2.3 met se = 0.8 en er zijn 13 vrijheidsgraden. Dan kan de nulhypothese


    1. FALSE: worden verworpen bij 0.02 maar niet bij 0.10.
    2. FALSE: worden verworpen bij 0.05 maar niet bij 0.02.
    3. TRUE: worden verworpen bij 0.02 maar niet bij 0.01.

    Solution

    Het correcte antwoord is:

    • worden verworpen bij 0.02 maar niet bij 0.01.
