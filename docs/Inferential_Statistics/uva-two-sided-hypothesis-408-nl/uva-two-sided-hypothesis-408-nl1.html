    Question

    Van twee steekproeven, die aselect uit twee populaties getrokken zijn, is het volgende bekend:
    Steekproef 1: n = 10, gemiddelde = 56, s.d. = 9
    Steekproef 2: n = 5, gemiddelde = 64, s.d. = 20
    Bij toetsen of de populatievarianties gelijk zijn is de nulhypothese


    1. FALSE: μ1 = μ2
    2. FALSE: s12 = s22
    3. FALSE: ¯X1 = ¯X2
    4. TRUE: σ12 = σ22

    Solution

    Het correcte antwoord is:

    • σ12 = σ22
