    Question

    Which analytical model is applied in an between-groups design with one categorical and one continuous independent variable and one dependent variable?
    The dependent variable is a continuous variable.


    1. FALSE: One-way ANOVA
    2. FALSE: Two-way ANOVA
    3. FALSE: MANOVA
    4. TRUE: ANCOVA

    Solution

    Een one-way ANOVA is een design met één categorische onafhankelijke variabele en één afhankelijke variabele.
    Een two-way ANOVA is een design met twee categorische onafhankelijke variabelen en één afhankelijke variabele.
    Een ANCOVA is een design met één categorische onafhankelijke variabele één continue afhankelijke variabele (covariaat) en één afhankelijke variabele.  
    Een MANOVA is een design met één categorische onafhankelijke variabele en meerdere afhankelijke variabelen.


    1. False
    2. False
    3. False
    4. True
