    Question

    Hieronder zie je een deel van de output van een mediatieanalyse met de Hayes Process.
    Met grote zekerheid kunnen we stellen dat het indirecte effect hoger is dan [1] … en kleiner dan [2] …. Dit betekent dat we [3] … bewijs hebben voor een mediatie effect in de populatie.


    1. TRUE: [1] = -0.0134 [2] = 0.0196 [3] = onvoldoende
    2. FALSE: [1] = 0.0036 [2] = 0.0081 [3] = voldoende
    3. FALSE: [1] = -0.0134 [2] = 0.0036 [3] = onvoldoende
    4. FALSE: [1] = 0.0036 [2] = 0.0196 [3] = voldoende
    5. FALSE: [1] = -0.0134 [2] = 0.0081 [3] = onvoldoende
    6. FALSE: [1] = 0.0081 [2] = 0.0196 [3] = voldoende

    Solution

    Dit is het bootstrap confidence interval van het indirecte effect (-0.0134, 0.0196). De 0 staat in het confidence interval. Dus, wij hebben onvoldoende bewijs dat er een mediatieeffect bestaat in de populatie.
