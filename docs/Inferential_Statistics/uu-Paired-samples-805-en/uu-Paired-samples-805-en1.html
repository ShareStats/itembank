    Question

    An eyeglass manufacturer is interested in finding out whether people can read a computer screen faster using glasses with specially coated lenses as opposed to glasses with regular (uncoated) lenses. Twenty people are asked to read a computer screen for 15 minutes with the coated lenses; on a separate day, the same 20 people are asked to read a computer screen for 15 minutes using lenses without the coating. The number of pages read by each participant for each condition was collected.

    Which statistical test should the researcher use?


    1. FALSE: t Test for Independent Means
    2. FALSE: t Test for a single sample
    3. TRUE: t Test for Dependent Means
    4. FALSE: Chi-Square Test for Independence
