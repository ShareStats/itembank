    Question

    Een onderzoeker is geïnteresseerd in het verschil in studieresultaten tussen studenten die optimistisch zijn versus studenten die pessimistisch zijn. Stel dat de onderzoeker geen significant verschil vindt tussen de twee groepen studenten, terwijl er in de populatie wel een verschil bestaat. wel bestaat.

    Wat is er aan de hand?


    1. FALSE: Er is een Type I fout gemaakt.
    2. TRUE: Er is een Type II fout gemaakt.
    3. FALSE: De nulhypothese is terecht verworpen.
    4. FALSE: De nulhypothese is terecht aangenomen.

    Solution

    Er is een type II-fout gemaakt.
    Bij statistische hypothesetests is een type I-fout de verwerping van een ware nulhypothese (ook bekend als een “fout-positieve” bevinding), terwijl een type II-fout het niet verwerpen van een valse nulhypothese is (ook bekend als een “fout-negatieve” bevinding). ook wel een “fout-negatieve” bevinding genoemd).
    Eenvoudiger gezegd, een type I fout is ten onrechte het bestaan van iets afleiden dat er niet is, terwijl een type II fout het niet verwerpen van een valse nulhypothese is. iets dat er niet is, terwijl een type II-fout het ten onrechte afleiden is van de afwezigheid van iets dat er wel is.
