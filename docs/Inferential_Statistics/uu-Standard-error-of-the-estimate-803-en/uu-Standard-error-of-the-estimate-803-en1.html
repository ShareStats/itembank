    Question

    A study examines whether there is a relationship between blood alcohol level and reaction time of young adults. The researcher performs a regression analysis on the collected data. The result is shown in the tables below.

    What does the Standard Error of the Estimate in the Model summary table mean?


    1. FALSE: This is the standard error of the slope coefficient
    2. FALSE: This is the standard error of the intercept
    3. TRUE: This is the standard deviation of the distances between the data points and the regression line
    4. FALSE: This is the standard deviation of the predictions we can make with this model
