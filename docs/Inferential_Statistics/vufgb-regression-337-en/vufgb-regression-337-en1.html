    Question

    The class Generalized linear models includes General linear models.

    Which link function belongs to the Generalized linear model?


    1. FALSE: Normal link
    2. TRUE: Identity link
    3. FALSE: Log link
    4. FALSE: Logit link

    Solution

    In the general linear model, the mean of y is modeled directly: g(μ)=μg(\mu) = \mu. This link function is called the Identity link (the link function consists of multiplying the average by 1).


    1. Incorrect
    2. Incorrect
    3. Incorrect
    4. Incorrect
