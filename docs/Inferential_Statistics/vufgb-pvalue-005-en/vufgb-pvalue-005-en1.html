    Question

    A sample statistic has a p-value of 0.63. What does this value indicate?


    1. TRUE: Probability for this result or more extreme is 63%, given that null hypothesis is true.
    2. FALSE: Probability for this result or more extreme is 63%, given that alternative hypothesis is true.
    3. FALSE: Based on this sample, the probability that the null hypothesis is true is 63%.
    4. FALSE: Based on this sample, the probability that the alternative hypothesis is true is 63%.

    1. Correct
    2. Incorrect
