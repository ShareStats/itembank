    Question

    Een wetenschapper voert een experiment uit om een speciaal soort kristal te maken. Hij gebruikt verschillende temperaturen om te zien welke temperatuur het sterkste kristal geeft. Hij zorgt ervoor dat de luchtvochtigheid in de kamer hetzelfde is voor alle kristallen. Welk type variabele is de luchtvochtigheid in de kamer?


    1. FALSE: De afhankelijke variabele
    2. FALSE: De onafhankelijke variabele
    3. FALSE: Een verstorende variabele
    4. TRUE: Een controlevariabele

    Solution

    De vochtigheid is een controlevariabele. De sterkte van het kristal is de afhankelijke variabele en de temperatuur de onafhankelijke variabele.
