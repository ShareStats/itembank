    Bij een covariantieanalyse (ANCOVA) wordt er eerst getoetst of er een interactie-effect is tussen de covariaat x en de nominale variabele z. Hiervoor wordt het volgende model geschat:

    E(y)=α+β1x+β2x+β3xzE(y) = \alpha+\beta_{1}x+\beta_{2}x+\beta_{3}xz

    Wat is bij deze toets de nulhypothese?


    1. FALSE: β1=β2=β3\beta_{1}=\beta_{2}=\beta_{3}
    2. FALSE: β1=0\beta_{1}=0
    3. FALSE: β2=0\beta_{2}=0
    4. TRUE: β3=0\beta_{3}=0

    1. Incorrect
    2. Incorrect
    3. Incorrect
    4. Correct
