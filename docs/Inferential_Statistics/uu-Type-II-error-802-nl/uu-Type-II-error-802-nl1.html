    Question

    Wat is bij het toetsen van een effect de betekenis van een Type II fout?


    1. FALSE: Het onterecht concluderen dat het effect significant is.
    2. TRUE: Het onterecht concluderen dat er geen significant effect is.
    3. FALSE: Het terecht concluderen dat het effect niet significant is.
    4. FALSE: Het terecht concluderen dat het effect significant is.

    Solution

    Een type II fout is het aannemen van H0H_0 terwijl in werkelijkheid deze verworpen had moeten worden.
