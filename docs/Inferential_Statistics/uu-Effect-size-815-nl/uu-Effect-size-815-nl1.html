    Question

    In een onderzoek naar de relatie tussen locatie van uitzending (Irak of Afghanistan) en depressiviteit van soldaten wordt in eerste plaats alleen maar gekeken òf de soldaten depressief zijn geworden, ja of nee. Een χ2\chi^2-toets wordt uitgevoerd en de variabelen bleken, in een steekproef van 60 soldaten, niet onafhankelijk: χ2(1,n=60)=5.88\chi^2(1, n = 60) = 5.88, p=.015p = .015. Wat is de waarde van de effectgrootte die hierbij gerapporteerd hoort te worden?


    1. FALSE: 0.0150.015
    2. FALSE: 0.0980.098
    3. TRUE: 0.3130.313
    4. FALSE: Dit is niet te bepalen omdat de kruistabel niet gegeven is.

    Solution

    Locatie is een dichotome variabele (Irak/Afghanistan), evenals depressie (ja/nee). Om in dit geval de effectgrootte uit te rekenen kan de phi-coefficiënt berekend worden:

    ϕ=χ2n=5.8860=0.313\phi= \sqrt{\frac{\chi^2}{n}} = \sqrt{\frac {5.88}{60}} =0.313
