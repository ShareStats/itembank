    Question

    De nulhypothese die wordt gebruikt bij hypothesetests met de Pearson correlatie is:


    1. TRUE: De correlatie is 0 in de populatie
    2. FALSE: De correlatie is niet gelijk aan 0 in de steekproef
    3. FALSE: De correlatie is niet gelijk aan 0 in de populatie
    4. FALSE: De correlatie is 0 in de steekproef

    Solution

    Hypothesen draaien altijd om de populatie, niet om je steekproef. En een nulhypothese verwacht geen effect/verschil/verband.


    1. Dit is het juiste antwoord!
