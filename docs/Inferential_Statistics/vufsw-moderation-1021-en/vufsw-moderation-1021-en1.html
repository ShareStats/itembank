    Question

    We can say that a moderation effect exists when the size of the relationship between the predictor and the outcome is:


    1. TRUE: Different for the different values of a third variable.
    2. FALSE: Equal to 0.
    3. FALSE: Not statistically significant.
    4. FALSE: The same for different values of the moderator.

    Solution

    Moderation means that the effect of X on Y is different at the various values of another variable M. Therefore the option " Different for the values of a third variable " is correct.


    1. True
    2. False
    3. False
    4. False
