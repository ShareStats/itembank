    Question

    Which of the following statements are correct?

    I. In a mixed-design ANOVA there is always one dependent variable
    II. In a mixed-design ANOVA there is always more than one variable that defines between group differences


    1. TRUE: Statement I is correct and statement II is incorrect.
    2. FALSE: Statement I is incorrect and statement II is correct.
    3. FALSE: Both statements are incorrect.
    4. FALSE: Both statements are correct.

    Solution

    There is always only one dependent variable in a mixed design ANOVA. So, statement I is correct.
    However, there can be one or more between variables. So, statement II is incorrect.
