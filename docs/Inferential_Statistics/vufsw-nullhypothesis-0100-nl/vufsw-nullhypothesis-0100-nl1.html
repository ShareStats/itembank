    Question

    Een onderzoeker beweert dat meisjes (groep 1) minder zakgeld krijgen dan jongens (groep 2). Hij toetst deze éénzijdige hypothese en hanteert een significantieniveau (alpha) van 5%.

    Hoe luidt in dit geval de nulhypothese?


    1. TRUE: H0: µ1 – µ2 = 0.
    2. FALSE: H0: µ1 ≠ µ2.
    3. FALSE: H0: µ1 + µ2 = 0.
    4. FALSE: H0: µ1 < µ2.
    5. FALSE: H0: µ1 = 0 en  µ2 = 0
    6. FALSE: H0: µ1 < 0 en µ2 = 0

    1. True
    2. False
    3. False
    4. False
    5. False
    6. False
