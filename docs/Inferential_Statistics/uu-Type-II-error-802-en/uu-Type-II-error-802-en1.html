    Question

    When testing an effect, what is the significance of a Type II error?


    1. FALSE: Falsely concluding that the effect is significant.
    2. TRUE: Falsely concluding that there is no significant effect.
    3. FALSE: Rightly concluding that the effect is not significant.
    4. FALSE: Rightly concluding that the effect is significant.

    Solution

    A type II error is the assumption of H0H_0 when in reality it should have been rejected.
