    Question

    Uit ieder van twee normaal verdeelde populaties met gelijke varianties wordt een steekproef getrokken.
    steekproef 1: n = 8, gemiddelde = 18, sd = 3.4
    steekproef 2: n = 6, gemiddelde = 24, sd = 6.9. Om te toetsen of de populatiegemiddelden gelijk zijn gebruiken we de gepoolde T-toets; de toetsingsgrootheid heeft hier een waarde van 2.155. Voor de bijbehorende tweezijdige P-waarde geldt dan


    1. FALSE: P < 0.01
    2. FALSE: 0.01 < P < 0.05
    3. TRUE: 0.05 < P < 0.10
    4. FALSE: P > 0.10

    Solution

    Het correcte antwoord is:

    • 0.05 < P < 0.10
