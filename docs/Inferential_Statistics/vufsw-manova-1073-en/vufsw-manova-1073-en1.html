    Question

    Which of the following is a test that can be used to assess multivariate differences in MANOVA?


    1. FALSE: Levene’s test
    2. FALSE: Box’s test
    3. TRUE: None of the other answers is correct
    4. FALSE: Mauchly’s test

    Solution

    The correct answer should be Pillai’s Trace, Hotelings T2, Wilk’s Lamda or Roy’s largest root. But none of these exists in the options.

    The Levene’s test is used to check the assumption of homogeneity of variances.

    The Box’s test is used to check the assumption of equality of covariance matrices and not the existence of multivariate differences.

    Mauchly’s test is used to check the assumption of sphericity in Repeated Measures


    1. False
    2. False
    3. True
    4. False
