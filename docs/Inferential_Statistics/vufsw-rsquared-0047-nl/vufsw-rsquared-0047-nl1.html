    Question

    Welke uitspraak over R2 is juist?


    1. FALSE: R2 = 1 indien de B-coëfficiënten van alle onafhankelijke variabelen in het regressiemodel de waarde 0 hebben.
    2. TRUE: De waarde van R2 kan nooit kleiner zijn dan 0.
    3. FALSE: R2 = 0 indien het regressiemodel geen enkele voorspellingsfout (‘prediction error’) maakt.
    4. FALSE: R2 = 0 indien er geen enkele correlatie is tussen alle onafhankelijke variabelen in het regressiemodel.
