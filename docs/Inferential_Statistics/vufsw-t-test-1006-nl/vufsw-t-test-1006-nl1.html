    Question

    Wat is de significantietest voor het testen van een b-coëfficiënt (t-test)?


    1. TRUE: Het toetst de nulhypothese dat, in de populatie, de coëfficiënt nul is nul is tegen de alternatieve hypothese dat de coëfficiënt in de populatie niet nul is
    2. FALSE: Het toetst de nulhypothese dat in de populatie de coëfficiënt gelijk is aan dezelfde waarde als die gevonden voor de steekproef tegenover de alternatieve hypothese dat de coëfficiënt in de populatie niet gelijk is aan dezelfde waarde als die gevonden voor de steekproef
    3. FALSE: Het test de nulhypothese dat de coëfficiënt niet nul is in de populatie tegen de alternatieve hypothese dat de coëfficiënt in de populatie nul is
    4. FALSE: Het test of de P-waarde van de coëfficiënt gelijk is aan 0,05.

    Solution

    De significantietest voor de b-coëfficiënt test het volgende:

    • 0 : b* = 0
    • a : b* ≠ 0
