    ANOVA is een multipele regressie met een kwantitatieve afhankelijke variabele en categorische (indicator) variabelen. De nulhypothese van een één-weg (één factor) ANOVAmet drie niveaus is dat de gemiddelden van alle drie de groepen (populaties) gelijk zijn. In de multipele regressie representeert de nulhypothese dat het vlak door de gemiddlelden van de drie groepen horizontaal is. De nulhypothese van de multipele regressie die equivalent is aan die van ANOVA is


    1. FALSE: H0: β1 = β2
    2. TRUE: H0: β1 = β2 = 0
    3. FALSE: H0: β1 = β2 = α

    1. H0: β1 = β2: Incorrect
    2. H0: β1 = β2 = 0: Correct
    3. H0: β1 = β2 = α: Incorrect
