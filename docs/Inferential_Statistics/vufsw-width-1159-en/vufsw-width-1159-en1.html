    Question

    A researcher is interested in the percentage of employees in the Netherlands who suffer from burn-out complaints. The researcher draws a random sample of 130 Dutch employees, for which we find: p̂ = 0.12

    Calculate the 95% confidence interval.


    1. TRUE: [0.064; 0.176]
    2. FALSE: [0.099; 0.141]
    3. FALSE: [0.091; 0.149]
    4. FALSE: [0.118; 0.122]

    Solution

    You can calculate this with the following formula: CI for p̂ +/- t(se) with se = (√(p̂(1 - p̂)) / n)

    Thus, se = (√ (0.12 * (1 - 0.12)) / 130 ) = 0.0285 and CI = 0.12 +/- 1.96 * 0.0285 = [0.064; 0.176]*
