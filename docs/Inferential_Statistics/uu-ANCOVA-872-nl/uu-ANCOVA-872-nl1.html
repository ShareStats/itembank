    Question

    Wat is een reden om een covariaat in je ANOVA op te nemen?


    1. FALSE: Om de binnen-groep error variantie te vergroten.
    2. FALSE: Omdat je geïnteresseerd bent in de interactie tussen de covariaat en de afhankelijke variabele.
    3. FALSE: Omdat je geïnteresseerd bent in de interactie tussen de covariaat en de onafhankelijke variabele.
    4. TRUE: Om de binnen-groep error variantie te verkleinen.

    Solution

    Een reden om een covariaat op te nemen in een ANOVA is dat binnen-groep error variantie kleiner wordt.
