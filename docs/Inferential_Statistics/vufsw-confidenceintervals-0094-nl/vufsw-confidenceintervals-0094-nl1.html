    Question

    Een onderzoeker voert een enkelvoudige regressie-analyse uit met “Inkomen” (netto maandsalaris, in euro’s) als afhankelijke variabele en variabele “Leeftijd” (leeftijd in jaren) als onafhankelijke variabele.

    Zij rekent een 95% betrouwbaarheidsinterval (confidence interval) uit van de slope. Dit interval blijkt [-0,02; 0,06] te zijn.

    Wat kunnen we hieruit concluderen, indien we tweezijdig toetsen of Leeftijd een significante invloed heeft op Inkomen? We hanteren hierbij een significantieniveau van 5% (α = 0,05).


    1. FALSE: Leeftijd heeft wel een significante invloed op Inkomen, want het interval bevat wel het getal 0,05.
    2. FALSE: Leeftijd heeft geen significante invloed op Inkomen, want het interval bevat wel het getal 0,05.
    3. TRUE: Leeftijd heeft geen significante invloed op Inkomen, want het interval bevat wel het getal 0.
    4. FALSE: Leeftijd heeft wel een significante invloed op Inkomen, want het interval bevat wel het getal 0.

    1. False
    2. False
    3. True
    4. False
