    Question

    Als leeftijd - wat een continue variabele is - een significant negatief moderatie-effect heeft op het effect van trainingsuren op sportprestaties, voor wie (d.w.z. mensen van welke leeftijd) is dan het effect van trainingsuren op sportprestaties het sterkst?


    1. FALSE: Het effect van training op prestaties varieert niet met de leeftijd
    2. TRUE: Jonge mensen
    3. FALSE: Mensen met een gemiddelde leeftijd
    4. FALSE: Oude mensen

    Solution

    A negative moderation effect means that the interaction effect is negative or, in other words, that the moderator M weakens the relationship between X and Y. Therefore, the option  ‘Young people’ is correct.


    1. False
    2. True
    3. False
    4. False
