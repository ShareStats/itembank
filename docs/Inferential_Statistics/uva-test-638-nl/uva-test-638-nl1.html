    2. FALSE: significant resultaat, verwerp H0 niet.

    4. TRUE: geen significant resultaat, verwerp H0 niet.

    1. significant resultaat, verwerp H0.: Incorrect
    2. significant resultaat, verwerp H0 niet.: Incorrect
    3. geen significant resultaat, verwerp H0.: Incorrect
