    Question

    Bij een experiment met drie verschillende rekenmethodes variëren de scores binnen elke experimentele groep, echter de gemiddelden per experimentele groep zijn aan elkaar gelijk. Bij uitvoering van een variantie-analyse (ANOVA) is dan


    1. TRUE: SSwithin > 0 en SSbetween = 0
    2. FALSE: SSwithin > 0 en SSbetween > 0
    3. FALSE: SSwithin = 0 en SSbetween > 0
    4. FALSE: SSwithin = 0 en SSbetween = 0

    Solution

    De zin ‘de scores binnen een experimentele groep variëren’ betekent SSwithin > 0. De zin ‘de gemiddelden per experimentele groep zijn aan elkaar gelijk’ betekent SSbetween = 0.
