    Question

    Een multipele regressie-analyse wordt uitgevoerd met drie explanatory variables. Er wordt collectief gekeken of met de drie variabelen de regressie een significant effect heeft op de voorspelling. Daarvoor wordt de nulhypothese opgesteld welke stelt dat H0: β1 = β2 = β3 = 0. De alternatieve hypothese, die het tegenovergestelde aangeeft van de nulhypothese, stelt dat


    1. FALSE: alle slopes (β’s) ongelijk aan 0 zijn.
    2. TRUE: er tenminste één slope (β) ongelijk aan 0 is.
    3. FALSE: alle slopes (β’s) verschillend zijn.

    1. alle slopes (β’s) ongelijk aan 0 zijn.: Incorrect
    2. er tenminste één slope (β) ongelijk aan 0 is.: Correct
    3. alle slopes (β’s) verschillend zijn.: Incorrect
