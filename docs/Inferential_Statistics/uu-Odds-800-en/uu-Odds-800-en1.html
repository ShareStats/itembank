    Question

    In a logistic regression model, with dependent variable Y, what does the value of odds mean?


    1. FALSE: The probability that Y = 1, or P(Y = 1).
    2. TRUE: The probability that Y = 1 versus the probability that Y = 0, or P(Y = 1) / P(Y = 0).
    3. FALSE: The probability that Y = 0 versus the probability that Y = 1, or P(Y = 0) / P(Y = 1).
    4. FALSE: The probability that Y = 0, or P(Y = 0).
