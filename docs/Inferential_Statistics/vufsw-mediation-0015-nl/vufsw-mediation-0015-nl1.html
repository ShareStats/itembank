    Question

    Hieronder staat het causale model bij een mediatie-analyse schematisch weergegeven.
    Wat is de beste omschrijving van het indirecte effect bij een mediatie-analyse?


    1. TRUE: Het indirecte effect is het effect van X op Y dat verloopt via de mediator (mediërende variabele).
    2. FALSE: Het indirecte effect is het effect van X op Y, gecontroleerd voor het effect van mediator (mediërende variabele) op Y.
    3. FALSE: Het indirecte effect is het effect van de mediator (mediërende variabele) op Y, gecontroleerd voor het effect van X op Y.
    4. FALSE: Het indirecte effect is de optelling van het effect van de mediator (mediërende variabele) op Y en van het effect van X op Y.
