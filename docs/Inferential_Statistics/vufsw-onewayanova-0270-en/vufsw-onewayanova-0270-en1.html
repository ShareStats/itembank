    Question

    Which analysis model is applied in a between-groups design with one categorical independent variable and one dependent variable?
    Here, the dependent variable is a continuous variable.


    1. TRUE: One-way ANOVA
    2. FALSE: Two-way ANOVA
    3. FALSE: MANOVA
    4. FALSE: ANCOVA

    Solution

    A one-way ANOVA is a design with one categorical independent variable and one dependent variable.
    A two-way ANOVA is a design with two categorical independent variables and one dependent variable.
    An ANCOVA is a design with one categorical independent variable one continuous dependent variable (covariate) and one dependent variable.
    A MANOVA is a design with one categorical independent variable and multiple dependent variables.


    1. True
    2. False
    3. False
    4. False
