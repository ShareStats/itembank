    Question

    Below is the formula of a single regression model of Y. ** **Y = a + b1 * X1 What does the “a” in the formula mean? And what value does it take?


    1. FALSE: This is the significance level, this value is determined by the researcher. and is usually 0.05.
    2. FALSE: This is the intercept, this value is 0 if X1 has no effect on Y.
    3. TRUE: This is the constant, this is the value of Y if X1 is equal to 0.
    4. FALSE: This is the alpha, this value belongs to a certain t-statistic and we have to look up in a table.
    5. FALSE: This is the value of the alternative hypothesis (Ha), this value is calculated by SPSS from the sample data.

    1. False
