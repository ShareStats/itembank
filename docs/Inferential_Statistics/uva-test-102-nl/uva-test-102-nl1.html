    Source df SS MS F
    Groups 3
    Error 25 20
    Total 28 35

    Gegeven is een significantieniveau van 0.05. Dan volgt uit de tabel dat


    1. FALSE: 4 ≤ F ≤ 5 en dus kan de nulhypothese van ANOVA worden verworpen.
    2. TRUE: 6 ≤ F ≤ 7 en dus kan de nulhypothese van ANOVA worden verworpen.
    3. FALSE: F ≤ 2 en dus kan de nulhypothese van ANOVA niet worden verworpen.
