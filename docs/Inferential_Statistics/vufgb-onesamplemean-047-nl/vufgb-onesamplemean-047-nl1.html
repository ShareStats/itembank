    Question

    Het gemiddelde in een steekproef van 25 personen is 15, de variantie 9.

    Bereken het 95% betrouwbaarheidsinterval voor het populatiegemiddelde.


    1. FALSE: 13.82 ; 16.18
    2. TRUE: 13.76 ; 16.24
    3. FALSE: 11.28 ; 18.72
    4. FALSE: 14.75 ; 15.25

    Solution

    95%BI:gemiddelde±t.025×se95\% \; BI:\;gemiddelde\pm t._{025}\times se

    Je moet dus eerst de t-waarde vinden en de standaardfout uitrekenen.

    t-waarde bij df=n−1=25−1=24:2.064df=n-1=25-1=24:\;2.064

    se=sn=35=0.6se = \frac{s}{\sqrt{n}} = \frac{3}{5} = 0.6

    Invullen geeft: 15±2.064×0.6→13.76;16.2415 \pm 2.064 \times 0.6 \rightarrow 13.76\; ;\; 16.24


    1. Incorrect
    2. Correct
    3. Incorrect
    4. Incorrect
