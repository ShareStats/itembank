    Question

    Stel, een onderzoeker heeft bij 130 werknemers van een groot bedrijf een vragenlijst afgenomen die in kaart bracht hoe zij hun leidinggevende beoordelen. De scores liggen tussen 1 (zeer negatief) en 10 (zeer positief). Hij wil nagaan of mensen met een vast contract [n=70] hun leidinggevende anders beoordelen dan mensen met een tijdelijk contract [n=60]. Hij berekent daarom het 95% betrouwbaarheidsinterval van het verschil tussen de gemiddeldes.

    Welke formule moet in dit geval gebruikt worden?


    1. FALSE: x̄  ± t ( se )
    2. FALSE: p̂  ± t ( se )
    3. FALSE:  ( x̄d ) ± t ( se )
    4. TRUE: ( x̄1 - x̄2  ) ± t ( se )

    1. False
    2. False

    4. True
