    Question

    You can find below the regression equation of a bivariate regression.

    Y = a + b1 * X1 

    What does “b” represent in this equation?


    1. FALSE: This is the significance level ‘alpha’ and it typically takes the value 0.05.
    2. FALSE: This is the slope and takes the value 0 when X1 has no effect on Y.
    3. FALSE: This is the slope and represents the predicted Y if X1  equals 0.
    4. TRUE: This is the slope and represents the predicted change in Y  if X1  increases with 1 unit.

    1. False
    2. False
    3. False
    4. True
