    Question

    Bij een ANOVA kan het verschil in gemiddelden worden beschreven in termen van regressie. Stel dat een éénweg ANOVA (één factor) met drie groepen wordt omgezet in een regressie met de volgende codering:

    groep x1 x2 gemiddelden interpretatie β
    1 1 0 μ1 = α + β1
    2 0 1 μ2 = α + β2
    3 0 0 μ3 = α

    De interpretatie van β3 is duidelijk, dat is α. De interpretatie van de andere twee coëfficiënten is als volgt:


    1. FALSE: β1 = μ1 - μ2 en β2 = μ1 - μ3.
    2. TRUE: β1 = μ1 - μ3 en β2 = μ2 - μ3.
    3. FALSE: β1 = μ2 - μ3 en β2 = μ1 - μ3.

    Solution

    Het correcte antwoord is:

    • β1 = μ1 - μ3 en β2 = μ2 - μ3.
