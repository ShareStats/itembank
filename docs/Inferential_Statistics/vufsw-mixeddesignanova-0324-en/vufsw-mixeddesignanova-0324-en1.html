    Question

    A researcher studies among young high school students the effect of a comprehensive school motivation program on truancy at start of a new school year, measured on a scale of 1-15. Scholars were divided into two groups: one group that attended and another who did not attend the program. The two groups are identical in all other characteristics relevant to truancy. Which method of analysis can the researcher * Not **use?


    1. FALSE: One-way ANOVA with truancy as the dependent variable and participation in the program as the independent variable
    2. FALSE: Linear regression truancy as dependent variable, participation to the program as the independent variable and all other measured characteristics as control variables
    3. TRUE: Mixed Design ANOVA with the scale of motivation as the dependent variable and with participation in the program as the within variable and all other measured characteristics as between variables
    4. FALSE: Linear regression with the scale of truancy as the dependent variable, participation in the program as the independent variable

    1. False
    2. False
    3. True
    4. False
