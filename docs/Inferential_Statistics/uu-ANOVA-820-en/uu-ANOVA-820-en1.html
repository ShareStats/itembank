    Question

    A main effect in a factorial design


    1. FALSE: Refers to any F-ratio that is significant in the ANOVA
    2. TRUE: Occurs when differences are found for the different levels of an independent variable
    3. FALSE: Occurs when the effect of one independent variable depends on the level of another independent variable
    4. FALSE: Is determined by calculating the row mean instead of the column mean

    Solution

    A main effect in a factorial design occurs when differences are found for the different levels of an independent variable
