    Solution

    Number of siblings is a stronger predictor of intelligence than Birth order. To decide which predictor is the stronger predictor, we need to look at the (absolute) value of the standardized regression coefficient. In model 2, the absolute value of the standardized regression coefficient of number of siblings (= -.233) is bigger than that of birth order (= .035).

    The statement Number of siblings strengthens the effect of Birth order is incorrect. This would be the case if the unstandardized regression coefficient of Birth order would have increased in model 2 compared to model 1, or if the p-value of the effect of Birth order would have decreased in model 2 compared to model 1. Neither is the case, so Number of siblings does not strengthen the effect of Birth order.
