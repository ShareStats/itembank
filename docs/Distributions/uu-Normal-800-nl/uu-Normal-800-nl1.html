    Question

    Er wordt een aselecte steekproef van n = 36 scores uit een populatie getrokken. Welke van de volgende verdelingen is normaalverdeeld?


    1. FALSE: De scores in de steekproef vormen een normale verdeling.
    2. FALSE: De scores in de populatie vormen een normale verdeling.
    3. TRUE: De verdeling van steekproefgemiddelden voor steekproeven van grootte n = 36 vormt een normale verdeling.
    4. FALSE: Geen enkele van de bovengenoemde verdelingen hoeft normaalverdeeld te zijn.
