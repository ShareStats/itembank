    Question

    For passing the theory driving exam, some learning styles are more successful than others. In research on the influence of learning styles on exam results, a separate test has been developed for each learning style. Two test developers talk to each other about the meaning of measurement error EE within classical test theory.

    Evaluate the following two statements:

    I. The measurement error E of a test does not correlate (rET=0r_{ET} = 0) with the true score TT. II. Between the measurement errors EE of the tests for the different learning styles is not consistent.


    1. TRUE: Pronunciation I and II are both correct.
    2. FALSE: Only pronunciation I is correct.
    3. FALSE: Only pronunciation II is correct.
    4. FALSE: Pronunciation I and II are both incorrect.

    Solution

    Theorem 1: According to classical test theory, measurement error is EE independent of the true score TT, because EE is an accidental and unsystematic measurement error.

    Theorem 2: Measurement error EE is accidental and unsystematic; there is no consistency among them.
