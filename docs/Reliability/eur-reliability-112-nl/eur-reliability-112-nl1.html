    Question

    Twintig parallelle items worden verwijderd uit een vragenlijst van 80 items die een alpha van .92 had. Wat wordt de nieuwe alpha? Gebruik de formule hieronder:

    Rxx−revised=n×RXX−original1+(n−1)RXX−originalR_{xx-revised} = \frac{n \times R_{XX-original}}{1+(n-1) R_{XX-original}}

    Merk op dat n = verlengingsfactor. Oorspronkelijk aantal items x verlengingsfactor = nieuw aantal items.


    Solution

    Om de nieuwe alfa te berekenen hebben we de Spearman-Brown formule nodig. In dit geval is n 6080=.75\frac{60}{80}=.75 Dus: Rxx−revised=.75×.921+(.75−1).92=.896R_{xx-revised} = \frac{.75\times.92}{1+(.75-1).92} = .896.
