    Question

    Gegeven is de onderstaande variantie-covariantie matrix van drie variabelen. Hoe hoog is Cronbach’s alpha?

    Variable V1 V2 V3
    V1 332
    V2 165 545
    V3 159 235 336

    Gebruik de volgende formule:

    Cronbach′sAlpha=(k/(k−1))×∑Cij/Sx2Cronbach's Alpha = (k / (k-1)) \times ∑C_{ij} / S^2_x


    1. FALSE: .21
    2. FALSE: .47
    3. TRUE: .72
    4. FALSE: .78

    Solution

    We calculate the sum of the covariance and double it: 2×(165+159+235)=2×559=11182 \times (165+159+235)=2 \times 559=1118. For the variance of the total score we add the item variances: 1118+332+545+336=23311118+332+545+336=2331. Cronbach’s alpha then becomes: 3÷2×1118÷2331=.723 \div 2 \times 1118 \div 2331=.72
