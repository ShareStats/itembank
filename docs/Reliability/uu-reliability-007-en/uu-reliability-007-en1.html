    Question

    Classical test theory is based on a thought experiment in which the same test is administered infinitely many times to the same respondent under similar conditions (replications).

    For a respondentâ€™s replications, then:


    1. FALSE: The test scores remain the same and the measurement error varies.
    2. FALSE: The test scores vary and the measurement error remains the same.
    3. TRUE: The true score stays the same and the measurement error varies.
    4. FALSE: The true score varies and the measurement error remains the same.

    Solution

    Test theory states that a respondent in the thought experiment will always have the same true score (X). However, his score on the test (T) will vary, which means that the measurement error also varies (e). The correct combination is given in answer option C.
